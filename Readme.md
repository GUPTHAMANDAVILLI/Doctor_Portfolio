# 🩺 Dr. John Doe — Portfolio Website

A modern, responsive portfolio website for **Dr. John Doe**, a board-certified cardiologist with over 15 years of experience in cardiovascular medicine.

## ✨ Features

- **Dark Premium Theme** — Elegant dark UI with blue/purple gradient accents
- **Glassmorphism Cards** — Frosted-glass style cards with backdrop blur effects
- **Animated Counters** — Stats animate into view (15+ years, 10,000+ patients, 98% satisfaction)
- **Scroll Reveal Animations** — Elements gracefully appear as you scroll down
- **Fully Responsive** — Optimized for desktop, tablet, and mobile devices
- **Contact Form** — Interactive appointment booking form with submission feedback
- **Smooth Navigation** — Fixed navbar with active section highlighting and smooth scrolling

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/GUPTHAMANDAVILLI/Doctor_Portfolio.git
   ```

2. **Open in browser**
   ```bash
   cd Doctor_Portfolio
   start index.html
   ```

   Or simply double-click `index.html` to open it in your default browser.

## 🛠️ Technologies Used

- **HTML5** — Semantic markup and page structure
- **CSS3** — Custom properties, glassmorphism, gradients, keyframe animations, and media queries
- **JavaScript (Vanilla)** — Intersection Observer API, animated counters, scroll effects, and form handling
- **Google Fonts** — [Outfit](https://fonts.google.com/specimen/Outfit) & [Inter](https://fonts.google.com/specimen/Inter)

---

## 📱 Responsive Design

The website is fully responsive across all screen sizes:

- **Desktop** (1024px+) — Full grid layouts with side-by-side hero
- **Tablet** (641px–1024px) — Two-column grids with stacked hero
- **Mobile** (≤640px) — Single-column layout with hamburger navigation
